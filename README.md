# NasdaQ-AI — نظام اكتشاف الأنماط المركّبة (NAS100)

نظام بحث إحصائي لاكتشاف واختبار متانة الأنماط المركّبة (composite patterns) المرتبطة بربح/خسارة صفقات استراتيجية تداول على `OANDA:NAS100USD`، باستخدام سجل الصفقات وبيانات الشموع بفاصل دقيقة واحدة. المنهجية مبنية على: عدم تسريب معلومات مستقبلية (no look-ahead)، التحقق بمنهجية walk-forward عبر السنوات، اختبار مونت كارلو لكل نمط على حدة، تصحيح المقارنات المتعددة (Benjamini-Hochberg FDR)، الأدلة البايزية، وفحوصات الثبات واعتماد النظام السوقي. التفاصيل والنتائج الكاملة موجودة في التقرير النهائي بالعربية (انظر أدناه).

## المتطلبات والتثبيت

```bash
pip install -r requirements.txt
```

Python 3.9+ مطلوب.

## التشغيل

```bash
python main.py
```

يشغّل هذا خط الأنابيب الكامل من البداية للنهاية (~90 ثانية) وينتج:
- `outputs/reports/final_report_arabic.md` — التقرير النهائي (سردي بالعربية، الكود وأسماء الأعمدة بالإنجليزية).
- `outputs/data/*.csv` — جداول الأنماط المرشَّحة والمُسجَّلة (تُعاد توليدها في كل تشغيل، غير متتبَّعة في git).
- `outputs/figures/*.png` — الرسوم البيانية (تُعاد توليدها في كل تشغيل، غير متتبَّعة في git).

يمكن أيضًا تشغيل أي وحدة منفردة للتحقق منها أو تصحيحها، مثال:

```bash
python -m src.pattern_search
python -m src.walk_forward
```

## هيكل المشروع

| المسار | الوصف |
|---|---|
| `analytics_1.csv` | سجل الصفقات (789 صفقة، 2020–2024) |
| `nasdaq_m1_2020_2024.parquet` | شموع دقيقة واحدة لنفس الأداة والفترة |
| `pattern-discovery-prompt.md` | المواصفة الأصلية للمشروع |
| `main.py` | تنسيق خط الأنابيب الكامل |
| `src/data_loading.py` | تحميل وتوصيف البيانات والتحقق من التزامن الزمني |
| `src/excursion.py` | إعادة بناء مسار MFE/MAE لكل صفقة من الشموع |
| `src/exit_quality.py` | تصنيف جودة الخروج والمبلغ المتروك على الطاولة |
| `src/regime_detection.py` | تصنيف النظام السوقي (اتجاه/تذبذب/تقلب) ونقاط التحوّل |
| `src/feature_engineering.py` | ميزات ما-قبل-الدخول (بدون أي تسريب مستقبلي) |
| `src/pattern_search.py` | البحث في فضاء الأنماط المركّبة |
| `src/walk_forward.py` | التحقق عبر نوافذ زمنية متوسّعة (سنة باختبار في كل مرة) |
| `src/monte_carlo.py` | اختبار متانة كل نمط عبر Bootstrap وإعادة التوزيع العشوائي |
| `src/bayesian_evidence.py` | الأدلة البايزية (Beta-Binomial، Bayes Factor) |
| `src/stability_regime.py` | ثبات الحدود واعتماد النمط على نظام سوقي محدَّد |
| `src/ml_shap.py` | طبقة LightGBM + SHAP تفسيرية فقط، عبر walk-forward |
| `src/shap_hypothesis_test.py` | اختبار موجَّه لميزات SHAP الأكثر ثباتًا كفرضية مُسجَّلة مسبقًا |
| `src/clustering.py` | تجميع الفائزين/الخاسرين وأطول سلسلة خسائر |
| `src/edge_decay.py` | تحليل تراجع الحافة عبر الزمن ونقاط التحوّل |
| `src/scoring.py` | Robustness Score المُجمَّع وترتيب الأنماط |
| `src/visualizations.py` | الرسوم البيانية |
| `src/reporting.py` | توليد التقرير النهائي بالعربية |
| `src/stats_utils.py` | أدوات إحصائية مشتركة (bootstrap، FDR، بايزي) |
| `src/config.py` | المسارات والثوابت والافتراضات الموثَّقة |
